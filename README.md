# 薪盡，火傳 · The Kindling

> 指窮於為薪，火傳也，不知其盡也。
> ——《莊子·養生主》
>
> 形體終將窮盡，如燃薪終將燒完；但火焰本身，會一直傳下去，沒有人知道它何時真正熄滅。

**The Kindling** 是一套把中國歷史上十一位文豪與政治家，還原成十一套完整認知框架（Agent）的計畫。每一位都不是「人設扮演」，而是根據其一手奏疏、文集、史論、書信重建出來的**決策協定**——他們怎麼想、怎麼吵架、怎麼在資訊不全時下判斷。木柴（他們的肉身與時代）早已燒盡，這個 repo 要接住的，是那團還在燒的火（他們的判斷力）。

十一人被設定為同一張議事桌上的十一張椅子，彼此有天然的死敵、盟友、鏡像與師生關係——這是一個**議會**，不是十一個孤立的聊天機器人。

一個完整的、以中國山水畫風格製作的展示網站（`index.html`）已包含在本 repo 中，內容包括每位議員的介紹、可下載的 Agent 檔案，以及一張視覺化的「議會關係軌道圖」。

每一位的完整定位、思維協定、說話口吻、21 世紀轉譯表與死敵矩陣，都寫在對應的 `skills/*_Agent.md` 裡；`index.html` 只展示濃縮版索引。

### 語料評級是什麼

評級標注這個 Agent 背後**一手史料的密度與完整度**，不是人物的重要性：

- **A**：一手著作、奏疏、書信、文集完整存世，可精確錨定觀點與句式。
- **A−／B+**：核心著作完整，但部分傳世內容經他人輯錄整理（如《貞觀政要》為吳兢所記），非全部出自本人手筆。
- **B+**：詩文史料豐富，但缺乏系統性的政論/奏疏原典。

每份 Agent 檔案開頭都有「語料誠實標注」段落，說明具體依據哪些原典，以及**防贗品底線**——避免把歷史人物寫成扁平化的刻板印象（例如蘇軾絕不能被寫成「無憂無慮的吃貨樂天派」）。

## 怎麼使用

### 方式一：單一議員，作為 Claude Skill

1. 從 `skills/` 下載你要的 `*_Agent.md`（或直接用 `index.html` 上的「下載 Agent」按鈕）。
2. 把檔案放進你的 Claude skills 目錄。
3. 直接在對話中召喚，例如：「請王安石看看這個改革方案」。他會依照檔案裡的思維協定、說話口吻與紅線來回應，而不是泛泛地扮演「一個叫王安石的人」。

### 方式二：把整份規範貼進系統提示

如果你的工具不支援 skills 目錄，可以直接把 `.md` 全文貼進 system prompt 或自訂指令欄位——每份檔案本身就是一份完整、可獨立運作的 spec，不依賴其他檔案。

### 方式三：多人議會（觀察化學反應）

把兩到三位有記載張力的議員同時召入同一個對話（例如王安石 × 司馬光、或范仲淹 × 王安石），直接丟一個真實的決策問題進去，看他們依各自的紅線互相反駁。每份 Agent 檔案的「六 · 議會張力與死敵矩陣」章節，記載了他和誰是天然死敵、誰是盟友、以及一段「對抗劇本」範例，可以直接當作 prompt 的起手式。

### 方式四：瀏覽網站

打開 `index.html`（純靜態，不需要建置工具），或部署到 GitHub Pages：

1. Repo → Settings → Pages
2. Source 選 `main` branch、根目錄
3. 存檔後幾分鐘內，網站會在 `https://edwson.github.io/The-Kindling/` 上線

網站支援繁體 / 简体切換（右上角），介紹每位議員、提供下載，並有一張議會關係軌道圖視覺化十一人之間的二十二組張力。

## 貢獻：新增一位議員

這個議會的椅子是留著的。如果你覺得少了誰，歡迎依同樣規範新增：

1. Fork 這個 repo。
2. 在 `skills/` 下新增 `你的人物_Agent.md`，格式請參照 [CONTRIBUTING.md](./CONTRIBUTING.md) 裡的完整規格（YAML frontmatter + 六大章節）。
3. 如果你也想讓他出現在網站上，在 `index.html` 的 `COUNCIL` 陣列裡補一筆資料（包含 slug、姓名、生卒、語料評級、代表語錄等），並在 `RELATIONS` 陣列裡補上他與既有議員的關係。整個網站是資料驅動的，補資料即可，不需要動版面代碼。
4. 提交 Pull Request，並附上你依據的一手史料來源。

新增議員最重要的門檻，是**語料誠實標注**——先誠實交代這個人的一手材料有多完整，再據此決定評級與防贗品底線，而不是先寫出一個好用的人設再回頭找史料背書。

## 免責聲明

這些 Agent 是基於一手史料重建的**決策協定與思維框架**，用於歷史研究、教育與決策推演的思想實驗，不代表對歷史人物的最終定論式還原，也不是嚴肅的史學考證成果。每份檔案中的「語料誠實標注」與「防贗品底線」章節，正是為了讓使用者清楚知道哪些是有一手史料支撐的、哪些是基於整體人格邏輯的合理推演。
